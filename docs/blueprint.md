# **App Name**: Shramik Hisab

## Core Features:

- Worker Profiles: Worker profile management (name, photo, daily wage, phone number)
- Attendance Tracking: Daily attendance tracking (present/absent, check-in/out time, per worker per day)
- Payment Overview: Display per-worker payments with current balances in red or green, depending on if money is due.
- Language Toggle: Option to switch language from UI (via toggle)
- AI Translator: AI-powered "tool" that translates the UI, updating both the display language as well as internal validation and display messages. Supports both Hindi and English.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to represent reliability and professionalism, avoiding any sense of levity.
- Background color: Light gray (#F0F2F5), providing a clean and neutral backdrop to improve content readability.
- Accent color: Soft orange (#FFAB40) for highlighting important actions and interactive elements without causing visual fatigue.
- Body and headline font: 'PT Sans' (sans-serif) to maintain simplicity and readability.
- Use clear, minimalistic icons for navigation and actions; filled icons preferred for better visibility.
- A clean, card-based layout will ensure workers, balances, and actions are grouped together.
